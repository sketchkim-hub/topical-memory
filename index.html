📌 Gemini API 키 발급 방법
1. Google AI Studio 접속
2. Google 계정으로 로그인
3. Create API Key 클릭
4. 생성된 키(AIza...로 시작)를 위 입력란에 붙여넣기
5. 저장 버튼 클릭 → 채점 시 Gemini가 사용됩니다

✅ Gemini API는 월 일정량까지 무료로 사용 가능합니다
